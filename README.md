# Arena-Battle-Game (Python)

Dieses Skript ist ein textbasiertes Kampfsystem, das ich geschrieben habe, um die Umsetzung von Vererbung und Listen-Logik in Python zu vertiefen.

### Überblick
- **Klassensystem:** Nutzung von Parent- und Child-Classes für verschiedene Helden (Slayer, Ironbreaker, etc.).
- **Kampflogik:** Rundenbasierte Abfrage von Angriffen und Spezialfähigkeiten.
- **Gegner-Verwaltung:** Über 20 Monster-Typen, die in Wellen über Listen-Iterationen aufgerufen werden.

### Stand der Entwicklung
- Grundgerüst der Kampfmechanik steht.
- Balancing der Klassen (HP/Schaden) ist implementiert.
- Fehlerbehandlung bei Benutzereingaben ist integriert.

### Ausführung
Einfach die `Final Version.py` mit Python 3 starten. Keine externen Bibliotheken nötig.
